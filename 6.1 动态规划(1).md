## 最长不下降子序列

* 原理
> f[i] = 1  
> f[i]=max{f[j]+1}（a[j]<=a[i] 且 j<i）


* 代码
```$c++
    # 第一种写法
    void code1{
        for(i = 0; i < n; i++) {
           dp[i] = 1;
           for(j = 0; j < i; j++){
            if(data[j] <= data[i] && dp[j] + 1 > dp[i])
                 dp[i] = dp[j] + 1;
           }
        }
    }

    # 第二种写法
   void code2{
        for(i = 0; i < n; i++) {
           dp[i] = 1;
           for(j = 0; j < i; j++){
            if(data[j] <= data[i])
                dp[i] = max( dp[i], dp[j] + 1 );
           }
       }
   }
```

## 最长公共子序列

* 原理
> 状态转移方程:  
>   dp[i][j]=dp[i-1][j-1]+1  
>   dp[i][j]=max(dp[i - 1][j] ,dp[i][j - 1])  

* 输入
> abcfbc  
> abfcab 
* 输出
> 4

* 代码
```$xslt
	for(int i=0;i<str1.size();i++){
		for(int j=0;j<str2.size();j++){
			if(str1[i] == str2[j]){
				dp[i+1][j+1]=dp[i][j]+1;
			}else{
				dp[i+1][j+1] = max(dp[i][j+1],dp[i+1][j]);
			}
		}
	}
```
